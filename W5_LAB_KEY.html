What is an effect size?

An effect size tells us how big a difference is on a standardized scale, so we can compare results across contexts.

Cohen’s (1988) effect size recommendations (Cohen’s D):

  • small ≈ 0.2
  • medium ≈ 0.5
  • large ≈ 0.8

These cutoffs are rules of thumb- not hard rules. The meaning of an effect always depends on research context!

movies_clean <- movies %>%
  filter(year >=2000) %>% 
  filter(str_detect(title, "^[A-Za-z]")) %>% 
  select(title, year, rating, length, Action:Romance) %>%
  mutate(
    action_comedy = case_when(
      Action == 1 & Comedy == 0 ~ "Action",
      Comedy == 1 & Action == 0 ~ "Comedy",
      TRUE ~ NA_character_
    )) %>%
  mutate(
    drama_comedy = case_when(
      Drama == 1 & Comedy == 0 ~ "Drama",
      Comedy == 1 & Drama == 0 ~ "Comedy",
      TRUE ~ NA_character_
    )) %>%
    mutate(
    romance_comedy = case_when(
      Romance == 1 & Comedy == 0 ~ "Romance",
      Comedy == 1 & Romance == 0 ~ "Comedy",
      TRUE ~ NA_character_
    )) %>%
  mutate(action_comedy = factor(action_comedy, levels = c("Action", "Comedy"))) %>%
  mutate(drama_comedy = factor(drama_comedy, levels = c("Drama", "Comedy"))) %>%
  mutate(romance_comedy = factor(romance_comedy, levels = c("Romance", "Comedy")))

Create 3 samples to compare ratings across each movie type comparison:

  1. Drop movies (rows) that were not of the specified genre or were not exclusive
  2. Group by genre (action_comedy, drama_comedy, romance_comedy)
  3. Randomly sample 200 movies from each group (N_total = 400)
set.seed(51234)  # Set seed so that everyone ends up with same samples

ac_movies <- movies_clean %>% 
  drop_na(action_comedy) %>%
  group_by(action_comedy) %>%
  slice_sample(n = 200) %>%
  ungroup()

dc_movies <- movies_clean %>% 
  drop_na(drama_comedy) %>%
  group_by(drama_comedy) %>%
  slice_sample(n = 200) %>%
  ungroup()

rc_movies <- movies_clean %>% 
  drop_na(romance_comedy) %>%
  group_by(romance_comedy) %>%
  slice_sample(n = 200) %>%
  ungroup()
