Hmmm… there is no function for that.

So let’s write a simple function ourselves!

# Step 1: Write your own function

get_modes <- function(x) {
  
  freq_table <- tabyl(x)
  
  freq_table |>
    filter(n == max(n)) |>
    pull(x)
}

# Step 2: Plug the input into your new function
